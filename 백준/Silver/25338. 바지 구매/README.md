<p>시루는 가을에 입을 바지를 미리 사기 위해 백화점에 왔다. 다리가 길고 저체중인 시루는 길이가 맞는 바지를 사면 허리가 너무 크고, 허리가 맞는 바지를 사면 길이가 짧아서 잘 맞는 바지를 찾지 못하고 있다.</p>
