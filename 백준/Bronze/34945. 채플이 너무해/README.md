### 문제 설명

<p>이화여대는 졸업을 위해 총 $8$번의 채플을 들어야 한다.</p>

<p>화연이는 그동안 채플을 단 두 번밖에 듣지 못했다. 그리고 이제 마지막 학기만 남은 그녀는 졸업을 해야 한다.</p>

<p>마지막 학기에 채플을 $x$번 성공적으로 들었다고 할 때, 화연이가 졸업할 수 있는지 판별하라.</p>

### 입력 

 <p>첫째 줄에 마지막 학기에 화연이가 성공한 채플 횟수를 나타내는 정수 $x$가 주어진다. ($ 0 \le x \le 7 $)</p>

### 출력 

 <p>화연이가 졸업할 수 있으면 <span style="color:#e74c3c;"><code>Success!</code></span>를 출력하고, 졸업할 수 없으면 <span style="color:#e74c3c;"><code>Oh My God!</code></span>을 출력한다.</p>
