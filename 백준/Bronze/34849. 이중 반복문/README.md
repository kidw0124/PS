### 문제 설명

<p>일반적인 컴퓨터는 $1$초에 대략 $1$억에서 $10$억 번 정도의 연산을 수행할 수 있다고 알려져 있다. 로하의 컴퓨터는 $1$초에 정확히 $1$억($10^8$)번의 연산을 수행할 수 있다.</p>

<p>로하는 자신의 컴퓨터에 $N$번 반복하는 반복문을 이중으로 겹친 프로그램을 실행하였다. 이 프로그램은 정확히 $N^2$번의 연산을 수행한다.</p>

<p>$N$이 주어질 때, 로하의 프로그램이 $1$초 이하의 시간 내에 모든 연산을 수행할 수 있을지 알아내라.</p>

### 입력 

 <p>첫째 줄에 반복문의 반복 횟수 $N$이 주어진다. ($1 \leq N \leq 100\,000$)</p>

### 출력 

 <p>첫째 줄에 로하가 실행한 코드가 $1$초 이하의 시간 내에 연산을 모두 끝낼 수 있다면 <code>Accepted</code>를, 아니라면 <code>Time limit exceeded</code>를 대소문자 구분 및 띄어쓰기에 맞게 출력한다.</p>
