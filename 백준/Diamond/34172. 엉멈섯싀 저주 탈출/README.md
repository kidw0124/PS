### 문제 설명

<p><a href="/problem/27300">엉엉이의 저주, 멈뭄미믜 저주, 섯섯시싀 저주</a>에 차례로 걸렸었던 현철이는 <a href="/problem/28696">섯섯시싀 저주</a>, <a href="/problem/31536">멈뭄미믜 저주</a>, <a href="/problem/32232">엉엉이의 저주</a>를 무사히 풀어내 끝난 줄 알고 안심하고 있었다. 그러나 모든 저주의 배후였던 세진이는 이런 현철이의 허를 찔러 새로운 저주인 <a href="/problem/34163">엉멈섯싀 저주</a>를 내렸다.</p>

<p style="text-align: center;"><img alt="" src="" style="width:600px;max-width:100%;"></p>

<p>현철이는 이제 MatKor Cup도 끝나는 마당에 저주에서 영원히 탈출하고 싶다. 이에 세진이는 현철이에게 제안을 하나 했다. 본인과 <strong>탈출을 건 한 판</strong>을 해서 이긴다면 영원히 저주를 풀어주기로 한 것이다.</p>

<p><strong>탈출을 건 한 판</strong>은 $N$개의 스케치북과 각 스케치북에 대응된 크레파스가 있으며, 각각의 크레파스로 최대 $L_1,L_2,L_3,\cdots ,L_N$의 길이를 그릴 수 있다. 처음에는 모든 스케치북에 아무것도 그려져 있지 않다. 현철이가 선공, 세진이가 후공으로 진행되며, 각자 턴을 번갈아가며 자신의 턴에 아래와 같은 행동을 한다.</p>

<ol>
	<li>현재 플레이어는 $N$개의 스케치북 중 하나와 아래 도형 중 하나를 선택한다.
	<ul>
		<li>원</li>
		<li>정삼각형</li>
		<li>정사각형</li>
	</ul>
	</li>
	<li>만약 해당 스케치북에 아무것도 그려져 있지 않다면, 외접원의 반지름이 $1$인 선택한 도형을 그린다.</li>
	<li>만약 해당 스케치북에 무엇인가 그려져 있다면, 현재 그려져 있는 그림을 모두 내부에 포함하도록 하는 최소 크기의 선택한 도형을 그린다.</li>
</ol>

<p>만약 현재 어떠한 스케치북을 골라도 남은 크레파스로 더 이상 어떠한 그림도 그릴 수 없다면 해당 플레이어는 패배한다.</p>

<p style="text-align: center;"><img alt="" src="" style="width:300px;max-width:100%;"></p>

<p>현철이와 세진이가 최적으로 플레이할 때, 누가 승리하는지 알아보자.</p>

<p>당연하지만, 누군가 패배하면 다른 한 명은 승리한다.</p>

### 입력 

 <p>첫 번째 줄에 테스트 케이스 $T(1\le T\le 1\, 000)$이 주어진다.</p>

<p>각 테스트 케이스 별로 첫 번째 줄에 $N(1\le N\le 1\, 000)$이 주어진다.</p>

<p>각 테스트 케이스 별로 두 번째 줄에 정수 $L_1,L_2,L_3,\cdots ,L_N(1\le L_i\le 300)$이 공백으로 구분되어 주어진다.</p>

### 출력 

 <p>각 테스트 케이스 별로 현철이가 승리하면 <span style="color:#e74c3c;"><code>No More Curse!</code></span>를, 세진이가 승리하면 <span style="color:#e74c3c;"><code>Curse will be forever!</code></span>을 한 줄에 출력한다.</p>
