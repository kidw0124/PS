### 출력 

 <p>첫 번째 줄에 목표를 달성하기 위한 최소 행동 횟수 <mjx-container class="MathJax" jax="CHTML" style="font-size: 109%; position: relative;"><mjx-math class="MJX-TEX" aria-hidden="true"><mjx-mi class="mjx-i"><mjx-c class="mjx-c1D43F TEX-I"></mjx-c></mjx-mi></mjx-math><mjx-assistive-mml unselectable="on" display="inline"><math xmlns="http://www.w3.org/1998/Math/MathML"><mi>L</mi></math></mjx-assistive-mml><span aria-hidden="true" class="no-mathjax mjx-copytext">$L$</span></mjx-container>을 출력한다. 만약 불가능한 경우 <span style="color:#e74c3c;"><code>-1</code></span>을 대신 출력한다.</p>

<p>만약 목표를 달성할 수 있다면, 두 번째 줄에 최소 횟수로 가열할 때 <mjx-container class="MathJax" jax="CHTML" style="font-size: 109%; position: relative;"><mjx-math class="MJX-TEX" aria-hidden="true"><mjx-mi class="mjx-i"><mjx-c class="mjx-c1D456 TEX-I"></mjx-c></mjx-mi></mjx-math><mjx-assistive-mml unselectable="on" display="inline"><math xmlns="http://www.w3.org/1998/Math/MathML"><mi>i</mi></math></mjx-assistive-mml><span aria-hidden="true" class="no-mathjax mjx-copytext">$i$</span></mjx-container>번째 금속 조각을 가열하는 횟수를 순서대로 공백으로 구분하여 출력한다. 가능한 방법이 여러 가지일 경우, 아무거나 출력해도 된다.</p>
