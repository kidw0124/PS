### 문제 설명

<p>하이비는 길이 $ N $의 순열 $ P $를 <a href="/problem/24652">선물로 받았다.</a> 하지만 받았던 수열이 마음에 들지 않았던 하이비는 아래 연산을 $ \left\lfloor \frac{3N}{2} \right\rfloor $회 이하로 사용해서 정렬된 순열, 즉 $ [1, 2, 3, \ldots, N] $을 만들기로 했다.</p>

<ul>
	<li>$ i $ $ l $ $ r $: $ P_i $를 $ \text{mex}([P_l, P_{l+1}, \ldots, P_r]) $로 변경한다. $ (1 \le l \le i \le r \le N) $
	<ul>
		<li>여기서 $ \text{mex}(A) $는 $ A $에 존재하지 않는 가장 작은 <strong>음이 아닌 정수</strong>를 의미한다. 예를 들면 $ \text{mex}([0, 3, 1]) = 2 $, $ \text{mex}([0, 1, 2, 3, 4]) = 5 $, $ \text{mex}([1, 3, 2]) = 0 $이다.</li>
		<li>연산을 적용한 뒤에 $ P $가 순열일 필요는 없다.</li>
	</ul>
	</li>
</ul>

<p>하이비를 위해 순열 $ P $를 정렬해 주자.</p>

### 입력 

 <p>첫째 줄에는 순열의 길이 $ N $이 주어진다. $( 1 \le N \le 5\,000 )$</p>

<p>둘째 줄에는 순열 $ P_1, P_2, \ldots, P_N $이 공백으로 구분되어 주어진다. $( 1 \le P_{i} \le N; $ $ P_{i} \neq P_{j} \text{ if } i \neq j )$</p>

### 출력 

 <p>첫째 줄에 사용한 연산의 횟수 $ K $를 출력한다. 연산 횟수를 최소화할 필요는 없다. $\left( 0 \le K \le \left\lfloor \frac{3N}{2} \right\rfloor \right)$</p>

<p>이후 $ K $개의 줄에 걸쳐 실행되어야 하는 연산을 한 줄에 하나씩 출력한다.</p>

<p>조건을 만족하는 모든 입력에 대해 답이 존재함을 증명할 수 있다.</p>
