### 문제 설명

<p>포켓몬을 사랑하는 민재는 요즘 모든 포켓몬을 포획하는 일명 도감 채우기를 하고 있다. 그러나 포켓몬마다 잡을 수 있는 확률이 존재한다.</p>

<p>포켓몬은 각각 체력이 존재하며, 최대 체력과 현재 체력에 따른 체력 계수 $h:=\frac{3\times HP_{max}-2\times HP_{now}}{3\times HP_{max}}$로 정의된다. 즉, 현재 체력이 최대 체력과 같다면, $h=\frac{1}{3}$이고, 현재 체력이 낮아질수록 $h$의 값은 $1$에 가까워진다. 또한 포켓몬별 고유 포획률 $r$, 볼의 종류 $b$, 상태 계수 $s$가 존재하여, 포켓몬을 잡을 수 있는 확률인 포획률 $p$는 다음과 같이 계산된다.</p>

<p>\[p:=\min\left\{ \frac{h\times r\times b\times s}{255} ,1 \right\}\]</p>

<p>순정파인 민재는 볼은 무조건 몬스터볼을 사용해 잡으려 하므로 $b=1$로 고정되며, 포켓몬의 초기 상태계수 $s=1$, 초기 체력 $HP_{now}=HP_{max}$이다.</p>

<p>민재는 매 턴 다음 행동 중 하나를 선택하여 할 수 있다.</p>

<ul>
<li>칼등치기(<span style="color:#e74c3c;"><code>F</code></span>)

<ul>
<li>$HP_{now}$가 $f$ 이하라면 $1$로 만든다.</li>
<li>$HP_{now}$가 $f$보다 크다면 $f$만큼 감소시킨다.</li>
</ul>
</li>
<li>뱀눈초리(<span style="color:#e74c3c;"><code>G</code></span>)
<ul>
<li>$s$가 $1$이라면 $s$를 $1.5$로 만든다.</li>
<li>$s$가 $1$이 아니라면 아무 일도 일어나지 않는다.</li>
</ul>
</li>
<li>하품(<span style="color:#e74c3c;"><code>Y</code></span>)
<ul>
<li>이번 턴은 아무 일도 일어나지 않는다.</li>
<li>다음 턴이 끝난 후 $s$가 $1$이라면 $s$를 $2.5$로 만든다.</li>
<li>다음 턴이 끝난 후 $s$가 $1$이 아니라면 아무 일도 일어나지 않는다.</li>
</ul>
</li>
<li>튀어오르기(<span style="color:#e74c3c;"><code>S</code></span>)
<ul>
<li>공격도 하지 않고 팔딱팔딱 튈 뿐 아무 일도 일어나지 않는다...</li>
</ul>
</li>
<li>포획하기(<span style="color:#e74c3c;"><code>C</code></span>)

<ul>
<li>현재 상태에서 몬스터볼을 던져 계산된 $p$의 확률로 포켓몬을 포획한다.</li>
</ul>
</li>
</ul>

<p>포켓몬은 한 번 포획에 실패하면 반드시 도망가게 되어 포획할 수 없다. 즉, 포획은 마지막 한 번만 가능하다.</p>

<p>도감을 빨리 채워야 하는 민재는 가장 확률이 높은 방법으로, 확률이 가장 높은 방법 중에서는 최소한의 행동으로 포켓몬을 포획해야 한다.</p>

<p>민재는 첫 턴의 행동을 하기 전 모든 행동을 미리 정한 후 행동한다.</p>

<p>최적으로 행동했을 때의 행동을 찾아 출력해 보자.</p>

### 입력 

 <p>첫 번째 줄에 테스트 케이스의 개수 $T(1\le T\le 1\, 000)$가 주어진다.</p>

<p>각 테스트 케이스의 첫 번째 줄에 잡고 싶은 포켓몬의 포획률과 최대 체력을 의미하는 정수 $r(1\le r\le 765)$과 $HP_{max}(1\le HP_{max}\le 1\, 000)$가 한 줄에 공백으로 구분되어 주어진다.</p>

<p>각 테스트 케이스의 두 번째 줄에 칼등치기의 데미지를 의미하는 정수 $f(1\le f\le 1\, 000)$가 주어진다.</p>

### 출력 

 <p>첫 번째 줄부터 $T$줄에 걸쳐 각 테스트 케이스 별로 한 줄에 최적으로 행동했을 때의 행동을 의미하는 <span style="color:#e74c3c;"><code>F</code></span>, <span style="color:#e74c3c;"><code>G</code></span>, <span style="color:#e74c3c;"><code>Y</code></span>, <span style="color:#e74c3c;"><code>S</code></span>, <span style="color:#e74c3c;"><code>C</code></span>로만 이루어진 문자열 $S$를 출력한다. 문자열의 마지막 문자는 <span style="color:#e74c3c;"><code>C</code></span>여야 하며, <span style="color:#e74c3c;"><code>C</code></span>는 문자열의 마지막을 제외하고 나타나면 안 된다.</p>
