### 문제 설명

<p>$-02493$. 누군가에게는 아무 의미가 없을 수 있지만, PD를 풀고 온 여러분에게는 $02493$의 부호를 뒤집은 수이다.</p>

<p>앞선 <a href="/problem/35146">PD : 02493</a>문제의 체커(유저가 출력한 정답이 맞는지 확인하는 코드)를 작성하던 동우는 또다시 큰 난관에 봉착했다. 동우는 유저가 출력한 정수 쌍 $\left( N,M,a,b \right)$에 대해 다음 문제를 $T$번 해결해야 한다.</p>

<ul>
<li>$i$행 $j$열의 원소 $A_{ij}$를 $\left( ai+bj \right)\bmod 10$으로 하는 $N\times M$ 모양의 $2$차원 배열 $A$를 생각한다.</li>
<li>여기서 그릴 수 있는 $\binom{N+1}{2}\times\binom{M+1}{2}$개의 직사각형들 중 직사각형 안의 숫자가 모두 다른 직사각형의 개수 $X$를 구해야 한다.</li>
</ul>

<p>그러나 테스트 케이스 $T$개를 처리하는 것은 너무 어려운 일이었다. 동우를 도와 주어지는 $T$개의 정수 쌍 $\left( N,M,a,b \right)$에 대해 문제를 해결해 보자.</p>

### 입력 

 <p>첫 번째 줄에 테스트 케이스의 개수 $T(1\le T\le 10^6)$가 주어진다.</p>

<p>두 번째 줄부터 $T$줄에 걸쳐 각 테스트 케이스별로 한 줄씩 유저가 출력한 정수 $N,M,a,b(1\le N,M,a,b\le 10^{6})$가 공백으로 구분되어 주어진다.</p>

### 출력 

 <p>첫 번째 줄에 각 테스트 케이스의 정답 $X$를 의미하는 정수 $X_1,X_2,\cdots ,X_T$를 공백으로 구분하여 출력한다.</p>
